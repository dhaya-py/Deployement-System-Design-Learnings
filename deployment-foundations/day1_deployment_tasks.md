 ## Learn definitions: DNS, IP, hosting, firewall

 ### DNS :

 DNS or Domain Name System is a internet's phone book that translates human-friendly domain name like (google.com) into machine-readable
 IP address, such as 192.168.1.1.  
 system connecting domain ‚Üí IP 

 ### IP :

 An IP address (Internet Protocol address) is a unique numeric lable assigned to each device connected to a computer network that uses
 Internet Protocol address for communication.
 
 it serves two main purposes, 
 1. Identifying the device on the network
 2. Locating the device to enable communication with other devices over the network like the internet.

 ### Hosting :

 Hosting is a service that stores a website's files on a powerful computer called a server, making them accessible to the internet.

 ### Firewall :

 Firewall is a networl security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
