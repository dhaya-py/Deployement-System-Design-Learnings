# Day 1 Deployment Notes

## What is deployment?

Deploying means running your code on a public server.

putting our code or application on a server so users can access it

## What is server?

A computer running our code or applicatin 24/7.
runs code continuously.

## What is domain?

Domain is human-friendly web address.
human-readable name. ex : google.com

## Key words to remember

DNS
Hosting
IP adress
firewall

## Concepts Learned
- Server: runs code continuously  
- Domain: website name  
- IP: numeric server address  
- DNS: system connecting domain → IP  

## Flow
Laptop → GitHub → Server → Internet
